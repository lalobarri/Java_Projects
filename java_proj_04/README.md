# 🚀 Introducción

La programación es una herramienta que permite resolver problemas del mundo real mediante el diseño de soluciones lógicas y estructuradas.

Sin embargo, conforme los sistemas crecen, también aumenta su complejidad. Por esta razón surge la **Programación Orientada a Objetos (POO)**, un paradigma que permite modelar elementos del mundo real mediante objetos, clases, atributos y métodos.

La POO facilita:

* ✅ La reutilización de código.
* ✅ El mantenimiento de aplicaciones.
* ✅ La organización de proyectos grandes.
* ✅ La escalabilidad de los sistemas.
* ✅ El trabajo colaborativo entre desarrolladores.

En esta práctica se desarrollará una aplicación capaz de calcular el área y perímetro de distintas figuras geométricas utilizando conceptos fundamentales como:

* 🧬 Herencia
* 🎭 Polimorfismo
* 🔒 Encapsulamiento
* 🏗️ Clases Abstractas
* 📦 Organización por paquetes

El objetivo es que el estudiante comprenda cómo transformar un problema real en una solución orientada a objetos utilizando Java.
