# 💻 ¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, robusto, multiplataforma y ampliamente utilizado en:

- 🌐 Desarrollo Web
- 📱 Aplicaciones móviles Android
- 🖥️ Sistemas empresariales
- ☁️ Servicios en la nube
- 🎮 Videojuegos
- 🤖 Sistemas inteligentes

Java permite desarrollar software profesional utilizando buenas prácticas de programación y una arquitectura escalable.

---

# 🧠 ¿Qué es la Programación Orientada a Objetos?

La Programación Orientada a Objetos (POO) es un paradigma de programación que organiza el software mediante objetos y clases.

La POO busca:

✅ Reutilizar código

✅ Organizar mejor los sistemas

✅ Facilitar el mantenimiento

✅ Simular problemas del mundo real

✅ Crear aplicaciones escalables

---

# 🔥 Principios Fundamentales de la POO

| Concepto | Descripción |
|---|---|
| 🧩 Clase | Plantilla para crear objetos |
| 🎯 Objeto | Instancia de una clase |
| 🔒 Encapsulamiento | Protección de datos |
| 🧬 Herencia | Reutilización de código |
| 🎭 Polimorfismo | Múltiples comportamientos |
| 🧠 Abstracción | Simplificación de procesos |

# ☕ Ejemplo Básico en Java

```java
public class Videojuego {

    String nombre;
    double precio;

    public Videojuego(String nombre, double precio) {
        this.nombre = nombre;
        this.precio = precio;
    }

    public void mostrarInfo() {
        System.out.println("Juego: " + nombre);
        System.out.println("Precio: $" + precio);
    }
}
```
